浅谈算法的时间复杂度与空间复杂度

发表于 2021-06-20 更新于 2021-07-04 分类于 技术分享

前言

在大学的教课书告诉我们，一个算法的优劣是通过它的复杂度来评判的。算法复杂度分为时间复杂度和空间复杂度：时间复杂度是指执行算法所需要的计算工作量；而空间复杂度是指执行这个算法所需要的内存空间。

复杂度一般使用大O表示法来表述，括号中一般使用 n 来表示数据规模。如果输入的数据为数组、字符串或集合，那么 n 代表它们的元素个数或长度，如果输入的数据为一个数字，那么 n 代表的数字大小。
