2021年度总结

发表于 2022-04-10 更新于 2022-04-15 分类于 生活分享
本文字数： 1.5k 阅读时长 ≈ 1 分钟

前言

2021年经历了很多事情，在这里做一个回顾和总结。（这篇年度总结本来是在今年过年的时候写的，但那时候光顾着玩，没想到拖延到了4月）

时间线

2月

从拱墅区的黄龙金茂悦小区搬家到西湖区的益乐新村南区

由于室友离开了，整租的房子只能违约中止合同了，赔了1.5个月的房租。

58同城真是初生，看房子全是中介，收我500的中介费还给我介绍个城中村，住的我差点抑郁了。

8平方米的房间，睡觉时脚都伸不直的，每天回家都像坐牢。我周末宁愿去公司，都不愿意呆在家里。

为数不多的优点是：离公司近，20min通勤；相对便宜的房租(1K)；便利的交通，出门就是丰潭路地铁和公交车站

但是让我选一次，我宁愿多花点钱也住个舒服的房子

5月

结束在杭州校宝在线的实习

虽然老大给我留了转正的坑位，但我还是选择离开实习了半年的公司。原因有3点：

一. 压抑的生活。当初跟我一起来的实习生（包括我的室友）都走了，我在公司里没有能说话的人（部门的同事比我大了一圈，没啥共同语言），感觉非常的孤独。加上牢房般的出租屋，让我每天都过得暗无天日

二. 有限的成长。当时做的项目是一个内部的运营系统，几乎全是CRUD的活，很难学到东西。实习期间虽然有安排导师，但是导师和我做的不是同一个业务，我们之间很少沟通。当然最大的问题还是我不够积极主动。虽然导师不太可能给我业务上的指导，但可以帮我做 code view，做好代码规范。

三. 也许有更好的。之前离职的室友后来进来大厂月薪上万，而我被老大压了薪资，没有达到期望的薪资。于是，想要出去试试，看看能不能找到待遇更好的 工作。

6月

入职杭州宇泛智能

从校宝离职后，我找了1星期多的工作，只收到了一些小公司的offer，大公司连面试都不给。待遇也都比上家差，有些甚至还是单双休。找工作找得我快怀疑人生了。后来面了宇泛智能，面试官感觉不错（因为前面遇到的面试官甚至让我感觉不到的尊重），给的offer薪资与上家相同，我也累了，就选择入职了。

公司是做物联网AIOT的，主打产品的考勤机/门禁和配套的Sass平台。

我所在的部门是智慧工地部门，主要的工作是跟客户当地的监管部门做对接，将考勤数据上报到监管的平台。

这份工作还是比较轻松的，我每天到点就跑了。

搬家到余杭区

离开了城中村后，就好像刑满释放了一般。

这次不再吝啬，选择余杭万达附件的房租1900/月的公寓。有厨房/独卫/大床的20多平米大房间，住的确实舒服。

绍兴文理学院毕业

没有大多的实感，好像来过又好像没有来过。

大学四年基本上是一个人窝在宿舍里开黑，虽然成绩还可以，也拿到过几次奖学金，但感觉少了点什么。

也许是一段社团活动的经历，也许是跟小女友热恋的甜蜜，也许是参加竞赛的奋斗，也许是跟舍友欢乐的日常。

我本可以做的更多，留下一些的美好回忆。我偶尔去想：如果选择做某件事，现在的我会不会更好。

不过往事已成定局，最重要的是现在和未来。

10月

试用期通过，转正成功

虽然转正答辩之前非常的紧张害怕，ppt改了又改，但实际还是很顺利的，感觉只是走个过场罢了。

总结

今年在租房上交了很多学费，买了3个教训：

  1. 在外工作租房非常重要，宁愿多掏点钱也别住得糟心（远离58同城）。
  2. 签合同的时候注意违约金，提前退租是要付违约金的。
  3. 如果室友不稳定，选择合租，不要选择整租。

对于初入职场的人，工作上要注意：

  1. 跟同事搞好关系，如果公司里连个说话的人都没有，上班跟上坟没什么区别
  2. 尽量去大点的平台，双休都没有的小公司线下面试都没有必要去
  3. 对于应届生来说，学历和八股文才是面试最重要的东西
  4. 试用期答辩只是一个形式，结果在一开始就定好了
