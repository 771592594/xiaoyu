使用Hexo+GitHub搭建博客

发表于 2021-06-13 更新于 2021-07-17 分类于 软件推荐

前言

大三的暑假，我用 Springboot 和 vue 手搓了一个博客网站并部署到了自己的云服务器上，但是用了一段时间后就吃灰了。因为自己造的实在是太难用了，最后还是回归 CSDN 写博客。

最近在网上了一些博客搭建的文章，发现很多人都使用 Hexo 这个静态博客框架，而且这个框架有很多好看的主题，配合 GitPage （github 提供给用户用来展示个人或者项目主页的静态网页系统）可以快速零成本的搭建博客。所以我也去学习了一下，花了一晚上的时间将博客部署到 GitPage 上。在这里做一个简单搭建的教程，给大家分享一下。

环境搭建

使用 hexo 搭建博客需要准备以下软件：

  • git（用于部署博客）
  • node.js（hexo的运行环境）
  • npm（js的包管理工具，node.js自带无需下载）
  • typora（markdown文本编辑器，用于写文章）

这些软件的安装都比较简单，在此不过多赘述。
